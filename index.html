传统色使用提示

色彩知识

把传统色放回使用场景里看：它先解决信息、材质和情绪，再成为画面里的颜色

取色顺序

先定角色

它是主色、辅色还是点缀？角色明确后，再选择色值、对比和使用媒介。

角色 主色 / 辅色 / 点缀 场景 屏幕 / 印刷 / 织物 校验 对比 / 光线 / 材质
五色线索

青、赤、黄、白、黑，可以帮你建立第一层秩序

青生发 / 清朗 赤醒目 / 热烈 黄中正 / 稳定 白留白 / 明净 黑收束 / 厚重
材质

别只看 HEX

矿物、植物、釉色和织物带来的质感不同。做品牌、课件或海报时，材质联想会影响颜色是否成立。

媒介

屏幕和印刷要分开判断

RGB 适合界面预览，CMYK 只是印刷沟通起点。落到实物前，仍要用光线、纸张和设备复核。
